<task type="auto">
  <name>Task 1: Create lib/realtime.ts channel registry</name>
  <files>lib/realtime.ts</files>
  <action>
Create `lib/realtime.ts` with these exports:

1. **`getOrCreateChannel(name: string): RealtimeChannel`** — checks a module-level `Map<string, RealtimeChannel>` for an existing channel with that name. If found, returns it. If not, creates via `supabase.channel(name)`, stores in the Map, and returns it. This handles React StrictMode double-mount (first mount creates, cleanup removes, second mount creates fresh).

2. **`removeChannel(name: string): void`** — looks up channel in Map, calls `supabase.removeChannel(channel)`, deletes from Map. No-op if name not found.

3. **`cleanupAllChannels(): void`** — iterates the Map, calls `supabase.removeChannel()` on each, then `channels.clear()`. Also call `supabase.removeAllChannels()` as a safety net after clearing the Map (catches any channels created outside the registry).

4. **`getChannelCount(): number`** — returns `channels.size`. Useful for debugging/testing.

Implementation details:
- Import `supabase` from `@/lib/supabase` and `RealtimeChannel` from `@supabase/supabase-js`
- The Map is module-level (singleton): `const channels = new Map<string, RealtimeChannel>();`
- Add `__DEV__` console.log statements: `[Realtime] Channel created: {name}`, `[Realtime] Channel removed: {name}`, `[Realtime] All channels cleaned up ({count} removed)`
- Do NOT subscribe the channel in this module — callers (hooks) chain `.on(...)` and call `.subscribe()` themselves

**Part B — Create migrations/035_realtime_publication.sql:**

```sql
-- Add content and likes tables to supabase_realtime publication
-- Required for postgres_changes subscriptions to receive events
ALTER PUBLICATION supabase_realtime ADD TABLE content;
ALTER PUBLICATION supabase_realtime ADD TABLE likes;
```

Note: If tables are already in the publication, ALTER PUBLICATION will error. To be safe, use:
```sql
DO $$
BEGIN
  IF NOT EXISTS (
    SELECT 1 FROM pg_publication_tables
    WHERE pubname = 'supabase_realtime' AND tablename = 'content'
  ) THEN
    ALTER PUBLICATION supabase_realtime ADD TABLE content;
  END IF;

  IF NOT EXISTS (
    SELECT 1 FROM pg_publication_tables
    WHERE pubname = 'supabase_realtime' AND tablename = 'likes'
  ) THEN
    ALTER PUBLICATION supabase_realtime ADD TABLE likes;
  END IF;
END $$;
```
