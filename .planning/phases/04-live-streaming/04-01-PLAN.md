<task type="auto">
  <name>Task 1: Create token-livekit Edge Function + lib/livekit.ts helper</name>
  <files>supabase/functions/token-livekit/index.ts, lib/livekit.ts</files>
  <action>
  1. Create `supabase/functions/token-livekit/index.ts` — a Deno Edge Function that:
     - Handles OPTIONS preflight with CORS headers (`Access-Control-Allow-Origin: *`, `Access-Control-Allow-Headers: authorization, x-client-info, apikey, content-type`)
     - Extracts Authorization header from request
     - Creates a Supabase client with the user's JWT to verify auth: `const { data: { user }, error } = await supabase.auth.getUser()`
     - Returns 401 JSON `{ error: "Unauthorized" }` if auth fails
     - Parses JSON body for `{ room_name: string, can_publish?: boolean }`
     - Returns 400 if room_name is missing
     - Creates an AccessToken using `import { AccessToken } from "npm:livekit-server-sdk"`:
       - identity: `user.id`
       - name: `user.user_metadata?.display_name || user.email || "user"`
       - ttl: `"2h"`
     - Adds grant: `{ roomJoin: true, room: room_name, canPublish: can_publish ?? false, canSubscribe: true }`
     - Calls `await at.toJwt()` (async, not sync toJWT)
     - Returns 200 JSON `{ token }` with CORS headers
     - Catches all errors and returns 500 JSON `{ error: err.message }` with CORS headers
     - Uses `import { createClient } from "npm:@supabase/supabase-js@2"` for Deno
     - Reads secrets: `Deno.env.get("LIVEKIT_API_KEY")`, `Deno.env.get("LIVEKIT_API_SECRET")`, `Deno.env.get("SUPABASE_URL")`, `Deno.env.get("SUPABASE_ANON_KEY")`

  2. Create `lib/livekit.ts` that:
     - Imports supabase from `./supabase`
     - Exports `LIVEKIT_URL = process.env.EXPO_PUBLIC_LIVEKIT_URL!` (already in .env)
     - Exports `async function fetchLiveKitToken(roomName: string, canPublish: boolean): Promise<{ token: string; serverUrl: string }>`
       - Gets session via `supabase.auth.getSession()`
       - Throws `"Not authenticated"` if no session
       - Calls `supabase.functions.invoke('token-livekit', { body: { room_name: roomName, can_publish: canPublish } })`
       - Throws on error
       - Returns `{ token: data.token, serverUrl: LIVEKIT_URL }`
  </action>
